# Quiz App

A simple interactive quiz application built with JavaScript, HTML, and CSS. Test your knowledge with multiple-choice questions.

## Features
- Multiple-choice questions
- Score tracking
- Simple and responsive design

## Technologies Used
- JavaScript
- HTML5
- CSS3

## How to Use
1. Open `index.html` in your browser.
2. Select an answer for each question.
3. Click "Submit" to see your score.
